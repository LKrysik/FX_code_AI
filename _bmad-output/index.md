## Project Summary

**FX Agent AI** is a cryptocurrency trading platform with pump & dump detection capabilities.

### Key Features
- Visual strategy builder (5-section condition system)
- 22+ custom trading indicators
- Backtesting and paper trading
- Live trading on MEXC Futures
- Real-time market monitoring

### Technology Stack
- **Backend:** Python 3.10+, FastAPI, QuestDB
- **Frontend:** Next.js 14, TypeScript, MUI, ReactFlow, Zustand

### Current State
- **Field Type:** Brownfield (existing codebase)
- **Tests:** 596 backend tests
- **Documentation:** 90+ existing docs

### Known Issues
- Trading/backtesting not fully functional
- UI has broken components
- Strategy builder incomplete
- Missing E2E process tests
